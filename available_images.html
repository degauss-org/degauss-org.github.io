Available DeGAUSS Images

image description version
ghcr.io/degauss-org/geocoder batch geocoding
ghcr.io/degauss-org/dep_index census tract-level deprivation index
ghcr.io/degauss-org/census_block_group census block group and tract FIPS
ghcr.io/degauss-org/st_census_tract spatiotemporal census tract FIPS 1970 - 2020
ghcr.io/degauss-org/roads proximity and length of major roads
ghcr.io/degauss-org/greenspace enhanced vegetation index
ghcr.io/degauss-org/drivetime distance and drive time to various care sites
degauss/schwartz_grid_lookup schwartz grid for spatiotemporal pollutant models
degauss/schwartz daily PM2.5, NO2, and O3 concentrations
ghcr.io/degauss-org/narr daily weather data (air temperature, humidity, etc)
ghcr.io/degauss-org/nlcd land cover data (imperviousness, land use, greenness)
ghcr.io/degauss-org/pm daily PM2.5
ghcr.io/degauss-org/aadt average annual daily traffic

The badges in the table above indicate the most recently released version of each DeGAUSS image. Clicking each badge links to a GitHub releases page with notes about changes made in each version release. It is possible to specify an older version by adding the tag onto the repo/image name, e.g. ghcr.io/degauss-org/geocoder:3.0.1.

The most recent version, as well as most older versions, of images prefixed with ghcr.io are avaialable via GitHub Container Registry. Some older versions are still available on Docker Hub.
